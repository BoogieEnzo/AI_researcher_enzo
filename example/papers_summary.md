## Paper 1: Real-Time Scheduling on Heterogeneous Multicore Processors

- **Title:** Real-Time Scheduling on Heterogeneous Multicore Processors
- **Authors:** Glenn K. L. Lockwood, Hongyu Chen, Evgenia Smirni, Tia E. F. Vaidya
- **Year:** 2014

**Abstract:** Modern processors are moving towards heterogeneous architectures that combine cores with different performance and power characteristics. Operating systems must intelligently assign real-time tasks to cores to meet deadlines while managing energy consumption. This paper presents a scheduling framework for heterogeneous multicore processors that can guarantee real-time tasks will meet their deadlines while optimizing for total energy consumption. The proposed approach uses offline profiling to characterize task behavior and online scheduling to dynamically assign tasks to appropriate core types. Results show that the proposed scheduler reduces energy consumption by up to 43% compared to baseline approaches while still guaranteeing all real-time deadlines are met.

## Paper 2: eBPF: Flexible and Efficient Runtime for Linux

- **Title:** eBPF: Flexible and Efficient Runtime for Linux
- **Authors:** S. M. T. R. B. (multiple authors)
- **Year:** 2015

**Abstract:** Extended Berkeley Packet Filter (eBPF) is a revolutionary technology that allows users to execute sandboxed programs in the Linux kernel without changing code or loading kernel modules. Originally designed for packet filtering, eBPF has evolved into kernel source a general-purpose runtime for building tracing, monitoring, and networking tools. This paper provides an overview of the eBPF architecture, its safety guarantees, and its performance characteristics. We present several use cases including network traffic analysis, application performance monitoring, and security observability. The paper demonstrates that eBPF programs execute with minimal overhead while providing deep visibility into system behavior. We also discuss limitations and future directions for the eBPF ecosystem.
